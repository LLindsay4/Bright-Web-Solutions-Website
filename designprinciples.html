Design Principles

The following resources have been provided to help you create websites that are clear, engaging, and easy to use.

They cover essential fundamentals like layout, colour, and navigation to ensure your designs communicate effectively and deliver a seamless user experience across all devices.

The Purpose


A website’s purpose defines its goals, audience, and content.

Every design choice should align with that purpose to clearly communicate the message and support user engagement.

Different Types of Websites

Usability


Usability ensures users can easily navigate and interact with a website.

Clear navigation, strong calls-to-action, and accessible information create a user-friendly experience, while testing helps fix any issues.

Usability 101

Typography


Typography affects a website’s readability and visual appeal.

Using consistent, well-chosen fonts, sizes, and spacing enhances design, accessibility, and brand identity.

The Typography Ultimate Guide

Colours


Colours can evoke emotions and create a visual hierarchy on a website.

Careful consideration should be given to the colour scheme to ensure it aligns with the brand identity and appeals to the target audience.

How to Choose Colors

White Space


Negative space refers to the empty space between design elements on a website.

Empty space between elements—reduces clutter, improves readability, highlights key content, and enhances a website’s visual appeal.

What is Whitespace?

Page Layout


The Layout of a website refers to how content is organised and displayed on each page.

A well-structured layout ensures that users can easily find the information they are looking for and navigate through the site efficiently.

How to Properly Layout A Website

Navigation


Navigation guides users through a website and helps them find information quickly.

Clear menus, breadcrumbs, and search improve usability and engagement. Well-structured navigation also boosts SEO and content discoverability.

Website Navigation Best Practice

Responsive Web Design


Responsive web design lets websites adapt to any device, ensuring a consistent user experience.

Using HTML, CSS, and JavaScript, layouts adjust dynamically, making sites seamless across desktops, tablets, and smartphones.

Guide to Responsive Web Design
