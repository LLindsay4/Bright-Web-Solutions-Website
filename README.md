# Bright-Web-Solutions-Website
Fictional business website created using HTML and CSS as part of the ELATT Software Development programme.

# Overview
The aim of the project was to develop a professional multi-page website while learning the fundamentals of HTML and CSS.

# Features
- Multi-page website
- Responsive layout
- Navigation menu
- Contact page
- Custom styling with CSS

# Technologies
- HTML5
- CSS3

# What I Learned
During this project I learned how to:
- Structure web pages using semantic HTML
- Style pages using CSS
- Create consistent page layouts
- Organise files within a web project
- Test and improve a website
